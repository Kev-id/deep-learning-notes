This repository serves as my centralized knowledge base and daily research tracker for **Multimodal Learning (MML)**. It contains deep-dive paper reviews, reproducible code snippets, framework alignment strategies, and environment troubleshooting notes.

---

## 📌 Research Interests
*   **Vision-Language Models (VLMs):** Cross-modal alignment, Contrastive Learning (CLIP-based variants), and Multimodal LLMs (e.g., LLaVA, BLIP).
*   **Video & Temporal Understanding:** Video-Language grounding, multi-frame tokenization, and efficient temporal modeling.
*   **Efficient Fine-Tuning & Evaluation:** Parameter-Efficient Fine-Tuning (PEFT/LoRA) on multimodal downstream tasks and robust metric evaluation (CLIP Score, CIDEr, etc.).
